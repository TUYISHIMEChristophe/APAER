# 🎓 APAER Institute Rusororo Website

A modern school website with an **Admin Panel** that allows updating content such as photos, news, announcements, and events dynamically using the browser.

---

## 🚀 Features

### 🌐 Frontend Website
- Home, About, Academics, Contact pages
- Media Center:
  - 📸 Gallery (Photos)
  - 🎥 Videos
  - 📰 News
  - 📢 Announcements
  - 📅 Events

---

### 🔐 Admin Panel (`update.html`)
- Upload **Photos from device**
- Add **News, Announcements, Events**
- Preview content instantly
- ✏️ Edit content
- ❌ Delete content
- Dark modern UI with animations

---

## ⚙️ How It Works

This project uses:

- **HTML, CSS, JavaScript**
- **localStorage (browser storage)**

### 📌 Important:
- All data is saved in the browser using `localStorage`
- No backend or database is used
- Data will be lost if:
  - Browser storage is cleared
  - Different device/browser is used
